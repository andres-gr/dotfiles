# arch-hyprland

Bare Hyprland compositor configuration for neo-dots.

This is a standalone, self-contained setup — no HyDE dependency.
For HyDE-specific configs, see `arch-hyde/`.

## Contents

- `.config/hypr/hyprland.conf` — main compositor config (all-in-one stub)
- `.config/hypr/hypridle.conf` — idle daemon config

## Status

STUB — functional base to boot into Hyprland.
Customize monitors, keybindings, and window rules for your setup.
