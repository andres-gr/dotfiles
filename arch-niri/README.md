# arch-niri

This stow package contains Niri + Dank Material Shell (DMS) configurations.

## Contents

- `.config/niri/` - Niri compositor configuration
- `.config/quickshell/` - Quickshell bar configuration
- `.local/share/dms/` - DMS-specific scripts (placeholder)

## Usage

This package is automatically selected when Niri is detected on your system.
